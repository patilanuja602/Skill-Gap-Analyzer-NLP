# AI Skill Gap Analyzer using Natural Language Processing (NLP)

## Overview

The AI Skill Gap Analyzer is an NLP-powered intelligent system designed to analyze candidate resumes, extract technical skills, compare them against industry job-role requirements, and identify missing competencies. The system helps students and job seekers understand their current skill profile and receive targeted recommendations for career development.

This project leverages Natural Language Processing (NLP) techniques using spaCy to perform automated skill extraction and gap analysis, reducing the need for manual resume evaluation.

## Problem Statement

Many students and job seekers struggle to understand whether their skills match industry expectations for a desired role. Traditional resume screening often relies on manual review, making it difficult to identify missing competencies efficiently.

The Skill Gap Analyzer addresses this challenge by automatically:

* Extracting skills from resumes
* Matching skills with job-role requirements
* Identifying missing skills
* Generating skill-gap reports
* Providing insights through visual analytics

## Performance Evaluation

The model was evaluated using standard classification metrics.

### Performance Metrics

| Metric    | Micro Average | Macro Average |
| --------- | ------------- | ------------- |
| Precision | 1.00          | 1.00          |
| Recall    | 0.8168        | 0.82          |
| F1-Score  | 0.8992        | 0.8914        |

### Interpretation

The results demonstrate high precision in identifying relevant skills while maintaining strong recall performance. The overall F1-score indicates effective balance between precision and recall, validating the reliability of the proposed Skill Gap Analyzer.
